What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming API for HTML and XML documents. It defines the logical structure of documents and the way a document is accessed and manipulated.

How do you select an element from the DOM?

The simplest and fastest way to select a single DOM element with JavaScript is by using its ID attribute. An ID is a unique identifier that I can assign to any HTML element using the id attribute.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation refers to the process of using event propagation or bubbling to handle events at a higher level in the DOM than the element on which the event originated. It allows us to attach a single event listener for elements that exist now or in the future.

How do you manipulate an element's attributes and styles using the DOM?

I can manipulate an element attributes by using the setAttribute() method. The setAttribute() method accepts two arguments. Also I can style an element by using getElementById() method. There we can add .styles and the name of the attribute name.
